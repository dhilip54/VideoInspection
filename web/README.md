# Real-Time Video Communication Web App

This is the ReactJS web client for the cross-platform real-time video communication app.

## Features
- Create or join a room
- Real-time video/audio using WebRTC
- Screen sharing
- Screenshot capture
- Toggle mic/camera, leave room

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the app:
   ```sh
   npm start
   ```

---

## Tech Stack
- ReactJS
- WebRTC
- Socket.IO
